# Emotion Detector

## Description
Emotion Detector is an AI-powered web application that analyzes user text and detects emotions such as joy, anger, sadness, fear, and disgust using the IBM Watson NLP library.

## Features
- Detects emotions from text input
- Returns dominant emotion
- Flask-based web interface
- Error handling for invalid inputs
- Unit tested application
- Static code analysis compliant

## Technologies Used
- Python
- Flask
- IBM Watson NLP
- Requests
- Pytest / Unittest
- Pylint
